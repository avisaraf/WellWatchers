Wellwatchers: Predicting the status of wells across Tanzania

Avi Saraf, Lily Zhang, Cindy Zhao

Download this project as a .zip file Download this project as a tar.gz file

Overview

We used a dataset from a Driven Data competition with Taarifa and the Tanzanian Ministry of Water to try to predict the status of wells in Tanzania. The goal of this project was to better understand which waterpoints would fail based on data about all of the wells in the country so as to improve maintenance operations for the wells and proactively improve access to water across the country.
